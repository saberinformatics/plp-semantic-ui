Future Now

Report pages generated with the semantic ui components have a sleek, subtle, stylishly flat, understated look that helps deliver a light and pleasant user experience. The components make world-class HTML5 reporting provided by semantic and jquery available in Pipeline Pilot.

screenshots

Reports

Drag to rearrange components in pipeline pilot to change report layout. Set component parameter values to specify exactly how the content will be shown (e.g. size, color, appearance). The resulting HTML reports will automatically adjust and reflow to match the viewer's browser window size or screen type.

Working with tabular data? Try the searchable Data Table component which packs powerful data navigation capabilities in HTML reports.

Web Apps

In addition to easily creating HTML reports you can also create web apps by using form element components such as text box, list box, button, and others. We added form validation options so that form elements will accept input values only in predefined ranges or format. In text boxes there is an active autosuggest available as users type. A configurable date picker is included. When users need to upload files while using a web app, an elegant multi-file upload component is available.

When the stock widgets are not enough, adding your own custom interactions on app pages is easy because we include jQuery on every page.
